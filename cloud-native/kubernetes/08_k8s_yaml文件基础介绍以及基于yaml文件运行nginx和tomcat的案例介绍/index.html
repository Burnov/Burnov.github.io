yaml文件基础与k8s架构下nginx+tomcat服务部署

Posted on 2022-01-18 In cloud native , kubernetes Valine:

yaml文件的基础介绍以及yaml文件运行nginx和tomcat的案例介绍

yaml文件的基本介绍

其实，我们的pod很多情况下都需要通过yaml文件来编写。基于yaml文件才能构建出公司也许需要的pod容器，因此，能够详细掌握yaml文件编写是能够正常管理号k8s集群的关键因素；

这里我们通过一个简单的项目部署来介绍yaml文件创建pod的过程；

yaml的格式介绍：

yaml文件的基本特点：

  • 大小写敏感
  • 使用缩进表示层级关系
  • 缩进时不容许使用tab键，只能容许使用空格键
  • 缩进的空格恕不不重要，只要相同层级的元素左侧对齐即可
  • 使用“#”表示注释，从这个字符一直到行尾都会被解析器忽略
  • 比json更加适用于配置文件

yaml与json格式的区别：

其实除了yaml文件意外，还有json文件也可以作为配置文件来写；但是json文件有一些缺陷：

  • json不能注释

  • json的可读性差

  • json的语法很严格比

但是json文件也由它自己的优点，例如：较适用于API返回值，也可用于配置文件，Jenkins就是使用json格式的配置文件；在网络上，由json和yaml文件的相互转换的功能；
