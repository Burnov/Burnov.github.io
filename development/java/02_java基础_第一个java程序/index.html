Path环境变量的配置

为什么需要配置path环境变量呢？javac指令不能再任意目录下运行，只能再Develop/jdk-8/bin目录下执行；

因为，如果java和javac工具只能再bin目录下运行的话，今后的代码都应该放在bin目录下才行，这样就非常不便于管理了；

如何配置呢？

第一步、找到”高级系统配置“->”环境变量“

第二步、环境变量->系统变量->新建

第三步、确定后，回到”系统变量“->选中”PATH“->编辑->编辑文本

这里需要注意的是，这里面所有的path路径默认都是以分号隔开的；所以，这里我们添加我们自己的java bin目录的路径，也是用分号隔开即可；

Hello World案例的常见问题

bug介绍：程序执行过程中的错误与漏洞，也就是代码写的不对的地方；有可能是语法问题，也有可能是逻辑问题。这里我们主要提到的是语法问题；作为一个程序员，我们需要有如下能力：

  • 具备有识别bug的能力 – 多看
  • 具备分析bug的能力 – 多思考，多查阅资料
  • 具备解决bug的能力 – 多尝试，多总结

HelloWorld程序中经常出现的错误：

  • string中的s小写
  • system中的第一个s小写

注意：JAVA程序是严格区分大小写的！

  • 少一个大括号，少一个小括号，少一个中括号
  • System.out.println(); // 后面没有“;”作为结束符号
