java语言介绍

语言：人与人交流沟通的表达方式

计算机语言：人与计算机之间进行信息交流的沟通的一种特殊语言

java语言：由美国sun（stanford university network）公司再1995年推出的计算机语言

java之父：詹姆斯高斯林

Java SE：标准版本，用于桌面应用开发，是其他两个版本的的基础；

​ 桌面应用：用户只用打开应用程序，程序界面会告知用户在最短的时间来熟悉软件的应用。同时主动带领用户完成他们的工作并得到最好的体验；（给Java EE打下基础）

Java ME：小型版本，用于嵌入式消费电子设备（已经被淘汰了）；

Java EE：web方向的网站开发；

​ 网页：用于数据展示（静态页面）

​ 网站：与后面的server有一个交互，展示出来是一个真实的数据；

java的夸平台原理

平台：Windows、Linux、Mac

跨平台：Java可以在任意的操作系统上运行；

​ java程序在运行的时候，都是在自己的java虚拟机的环境中运行的。所以，只需要在对应的操作系统上面安装对应的java虚拟机即可运行相同的java程序。对于客户而言，java程序是不变的，且在任意平台都可以正常执行；

​ 因此说java程序是可以跨平台的；

JDK：Java Development Kit,Java软件开发工具包，内部包含了编译和运行的工具；
