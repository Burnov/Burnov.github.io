注释

在指定的位置输入说明信息

  • 单行注释：// 注释信息
  • 多行注释：/* 注释信息 */
  • 文档注释：/** 注释信息 **/

目前还用不上文档注释信息的内容；

关键字

被Java赋予了特定意义的英文单词，就是我们Java程序中的关键字；例如之前在halloworld程序中使用到的public、class都是属于关键字，它们都有自己特定的含义；

  • public – 限制类名需要和文件名一致；
  • class – 用于创建一个类；

Java程序中的关键字字母都是小写，总共有50多个；这里不会一一介绍，我们需要在代码中学习，再总结；再notepad++编辑器中，关键字一般显示为紫色：

常量

在程序执行的过程中，不会变的值就是常量；

常量的分类：

  • 字符串常量：被双引号包裹的，就是字符串常量；
  • 整数常量：整数
  • 小数常量：小数
  • 字符常量：被单引号包裹的单个字符，只能是单个字符；
  • 布尔常量：true、false（单独打印是没有任何意义的），为后面的逻辑运算打下基础；
  • 空常量：null

变量介绍

为什么需要有变量呢？

在我们常用的环境中，在输入用户名和密码的时候，往往都是不固定的；服务端接收的是所有人的用户名和密码，这时候就需要有一个空间来存储用户输入进来的该用户名和密码，方便进行下一步的处理；

而，内存中存储这些数据的空间，需要被命名，这样才方便调用。这些个名字，我们就叫做变量；

注意：首先变量运行于内存中，其次就是浏览器提交过来的数据也是先存储于内存中，最后存储过来的数据都以变量名的方式指定，它就是一个“门牌号码”；

结论：变量就是内存中的存储空间，空间中存储着经常发生的变化的量（变量数据）；

Java程序属于强类型语言，对于每一种数据都给出了明确的数据类型，也分配了不同的内存空间，所以他们表示的数据大小也是不一样的；

数据类型 关键字 byte 大小范围
整数 byte 1 -128 ~ 127
short 2 -32768 ~ 32767
int 4 -2的31次方 ~ 2的31次方
long 8 -2的63次方 ~ 2的63次方
浮点数 float 4 1.401298e-45 ~ 3.402823e+38
double 8 4.9000000e-324 ~ 1.797693e+308
字符 char 2 0 ~ 65535
布尔 boolean 1 true,false

说明：e+38表示以10的38次方，e-45表示为10的负45次方

总结：数据类型

  • 基本数据类型
    • 数值型
      • 整数（1、byte ; 2、short； 3、int； 4、long；）
      • 浮点数（1、fload； 2、double；）
      • 字符（char）
    • 非数值型
      • 布尔值（boolean）
  • 引用数据类型（后面讨论）
    • 类（class）
    • 接口（interface）
    • 数组（[]）

标识符

标识符表示的就是变量、方法以及类的名字；我们统称为标识符；它有如下规则：

  • 由数字、字母、下划线以及$组成
  • 不能以数字开头
  • 不能是关键字
  • 区分字母大小写

如：

常见的命名约定：

  • 约定1：标识符是一个单词的时候，首字母小写。如：name
  • 约定2：标识符是多个单词的时候，第一个单词首字母小写，其他单词首字母大写，如：firstName。这种命名方式，我们也叫做小驼峰命名法；
  • 约定3：标识符也可以首字母大写，子啊有多个单词的时候，如：AllStudent。大驼峰命名法；
