java语言介绍

语言：人与人交流沟通的表达方式

计算机语言：人与计算机之间进行信息交流的沟通的一种特殊语言

java语言：由美国sun（stanford university network）公司再1995年推出的计算机语言

java之父：詹姆斯高斯林

Java SE：标准版本，用于桌面应用开发，是其他两个版本的的基础；

​ 桌面应用：用户只用打开应用程序，程序界面会告知用户在最短的时间来熟悉软件的应用。同时主动带领用户完成他们的工作并得到最好的体验；（给Java EE打下基础）

Java ME：小型版本，用于嵌入式消费电子设备（已经被淘汰了）；

Java EE：web方向的网站开发；

​ 网页：用于数据展示（静态页面）

​ 网站：与后面的server有一个交互，展示出来是一个真实的数据；

java的夸平台原理

平台：Windows、Linux、Mac

跨平台：Java可以在任意的操作系统上运行；

​ java程序在运行的时候，都是在自己的java虚拟机的环境中运行的。所以，只需要在对应的操作系统上面安装对应的java虚拟机即可运行相同的java程序。对于客户而言，java程序是不变的，且在任意平台都可以正常执行；

​ 因此说java程序是可以跨平台的；

常量

在程序执行的过程中，不会变的值就是常量；

常量的分类：

  • 字符串常量：被双引号包裹的，就是字符串常量；
  • 整数常量：整数
  • 小数常量：小数
  • 字符常量：被单引号包裹的单个字符，只能是单个字符；
  • 布尔常量：true、false（单独打印是没有任何意义的），为后面的逻辑运算打下基础；
  • 空常量：null

变量介绍

为什么需要有变量呢？

在我们常用的环境中，在输入用户名和密码的时候，往往都是不固定的；服务端接收的是所有人的用户名和密码，这时候就需要有一个空间来存储用户输入进来的该用户名和密码，方便进行下一步的处理；

而，内存中存储这些数据的空间，需要被命名，这样才方便调用。这些个名字，我们就叫做变量；

注意：首先变量运行于内存中，其次就是浏览器提交过来的数据也是先存储于内存中，最后存储过来的数据都以变量名的方式指定，它就是一个“门牌号码”；

结论：变量就是内存中的存储空间，空间中存储着经常发生的变化的量（变量数据）；

标识符

标识符表示的就是变量、方法以及类的名字；我们统称为标识符；它有如下规则：

  • 由数字、字母、下划线以及$组成
  • 不能以数字开头
  • 不能是关键字
  • 区分字母大小写

如：

常见的命名约定：

  • 约定1：标识符是一个单词的时候，首字母小写。如：name
  • 约定2：标识符是多个单词的时候，第一个单词首字母小写，其他单词首字母大写，如：firstName。这种命名方式，我们也叫做小驼峰命名法；
  • 约定3：标识符也可以首字母大写，子啊有多个单词的时候，如：AllStudent。大驼峰命名法；
